## Examples of Building AI Agents with Ollama, LangChain, and Gradio

- **Chat Memory**  
  A simple conversational agent that preserves chat history (no external tools).  
  *Uses:* Ollama, LangChain, Gradio

- **ReAct Agent with Search**  
  An agent that integrates the SerpAPI search tool for real-time information retrieval, while maintaining conversation history.  
  *Uses:* Ollama, LangChain, Gradio, SerpAPI

- **Job Application Assistant**  
  An agent that helps tailor your CV and cover letter based on a given job description.  
  *Uses:* Ollama, LangChain, Gradio
